# 힘마법사 키우기 (Power Mage Rising)

> 귀엽고 잔인한 방치형 RPG — 박살 쾌감과 확실한 성장의 설계

## 🎮 게임 개요
| 항목 | 내용 |
|------|------|
| 장르 | 방치형(Idle) RPG |
| 플랫폼 | 모바일 (Android / iOS) |
| 엔진 | Unity 6000.3.6f1 / URP 2D |
| 아트 | Pixel Art + AI VFX |
| 사운드 | AI SFX 14종 (ElevenLabs) + BGM (Suno AI) |

## ✅ 구현 완료 시스템
- 자동 전투 + 플레이어 이동 (최근접 적 추적, 잔상 이펙트)
- 궤도형 스킬 4종 (Staff Rush / Meteor Rain / Chain Lightning / Berserker Dash)
- 수동 공격 버튼 (ATK×10)
- 사운드 시스템 (AudioManager 싱글톤, SFX 풀링 8채널, 빈도 제어)
- 웨이브/스폰 시스템 (적 5종, 10~22마리)
- 보스 시스템 (3종 + FixedGoldBonus)
- 레벨업 + 실패 없는 강화 (HP 2차 공식)
- 오프라인 보상 (최대 8시간)
- 전투 피드백 (히트스톱 + 혈흔 + 카메라 흔들림)
- 메인 화면 (ChatGPT 일러스트 + BGM)
